# Atividade: Conexão Python com NoSQL (MongoDB e Redis)

Este repositório contém a resolução da atividade prática de integração entre Python, MongoDB Atlas e Redis Cloud. 
O script realiza operações de CRUD em ambos os bancos e implementa um sistema de cache simples.

## Tecnologias Utilizadas
- **Python 3**
- **MongoDB Atlas** (Banco de dados de Documentos)
- **Redis Cloud** (Banco de dados em Memória/Chave-Valor)
