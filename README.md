# Atividade: Conexão Python com NoSQL (MongoDB e Redis)

Este repositório contém a resolução da atividade prática de integração entre Python, MongoDB Atlas e Redis Cloud. 
O script realiza operações de CRUD em ambos os bancos e implementa um sistema de cache simples.

## 🚀 Tecnologias Utilizadas
- **Python 3**
- **MongoDB Atlas** (Banco de dados de Documentos)
- **Redis Cloud** (Banco de dados em Memória/Chave-Valor)
- **VSCode** (Ambiente de Desenvolvimento)

## 📋 Pré-requisitos
Antes de executar o código, certifique-se de ter o Python instalado na sua máquina e crie as contas gratuitas no [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) e no [Redis Cloud](https://redis.com/try-free/).

## 🔧 Instalação das Dependências

Abra o terminal do seu projeto e instale as bibliotecas necessárias executando o comando abaixo:

```bash
pip install pymongo redis python-dotenv
